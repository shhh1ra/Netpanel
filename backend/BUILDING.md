# Packaging backend

The release app ships the Python API as a Tauri sidecar.

Windows sidecar name:

```text
frontend/sidecars/cisco-backend-x86_64-pc-windows-msvc.exe
```

Linux sidecar name:

```text
frontend/sidecars/cisco-backend-x86_64-unknown-linux-gnu
```

Both files are produced by the scripts in `scripts/`.
